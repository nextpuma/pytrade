## Title

# Pytrade

## Desctiption

A cryptocurrency trading bot written in python

## Usage

```
usage: pytrade.py [-h] [-b | -l] [-s SYMBOL] [-i INTERVAL] [-g] [-v]
                  [-S STARTTIME] [-d]

This is PYTRADE

optional arguments:
  -h, --help            show this help message and exit
  -b, --backtest        Backtest some strategies
  -l, --live            Live trading
  -s SYMBOL, --symbol SYMBOL
                        This is the symbol you wish to trade
  -i INTERVAL, --interval INTERVAL
                        The interval for the trades. Defaults to 1m
  -g, --graph           Whether to graph the result
  -v, --verbose         Verbose output from backtests
  -S STARTTIME, --startTime STARTTIME
                        How long ago to backtest from e.g 1 week ago
  -d, --debug           Enable debug output
```
